# Project Title: Security Incident Reporting System

# Degree Program: Bachelor of Science in Cyber Security and Digital Forensics Engineering

# Group Number: Group 06

## Project Description:

This is a specialized web-based system designed to allow users to record, track, and manage cyber security incidents. The application allows users to report new incidents, view a comprehensive list of all incident reports, and search for specific cases using a unique Incident ID. This tool is essential for security teams to perform digital forensics investigations and strengthen organizational security posture.
